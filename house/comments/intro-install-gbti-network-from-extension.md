We ship the GBTI MCP server right inside our Chrome extension folder so you do not have to install anything extra to author with Claude Code. The browser never runs that file; Claude Code reads it from the folder and runs it for you. We built this guide so you can point Claude Code at it, sign in once, and then publish posts, products, and prompts through our pull-request gate, even with your browser closed. If you already have our extension, you are most of the way there.
