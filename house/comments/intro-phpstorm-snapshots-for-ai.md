We built Snapshots for AI because sharing your code with LLMs should not be tedious. This plugin for JetBrains IDEs lets you quickly generate clean markdown snapshots of your files and project structure, then drop them into ChatGPT, Claude, or whatever AI tool you are using. If you work with AI assistants to debug, refactor, or document code, this is built for you.
