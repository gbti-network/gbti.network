We built Snapshots for AI because we kept hitting the same friction point: sharing code context with AI assistants meant manual copying, pasting, and formatting. This extension does that work for you. One click in VS Code creates a clean, AI-ready markdown snapshot of your codebase. It is for anyone working with AI tools daily, whether you are debugging with Claude, brainstorming architecture with ChatGPT, or using AI for code review. We designed it to be fast and simple: select your files, hit the camera icon, and your context is ready to paste into any AI chat.
