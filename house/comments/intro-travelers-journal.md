We built Travelers Journal because server administrators asked for a simple way to communicate with players without leaving the game. This Minecraft Fabric mod lets you create a shareable in-game book that pulls its content from a markdown file you control. Update the file, and every copy updates instantly across your server. We designed it for communities that want to onboard new players, share rules, or keep everyone on the same page. Check it out if you run a Minecraft server.
