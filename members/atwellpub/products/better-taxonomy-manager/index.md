## Advanced Taxonomy Manager

Advanced Taxonomy Manager brings a modern interface to WordPress taxonomy management.

It is now completely free on the [WordPress plugin directory](https://wordpress.org/plugins/better-category-manager/). The earlier category-only tool (Better Category Manager) was merged into it, so the full drag-and-drop taxonomy experience that was once a network perk is now available to everyone at no cost.

## Features

-   Manage priority and nesting through drag and drop UI capabilities.
-   Enjoy an improved UI that leverages AJAX to avoid hard refreshes
-   Quickly export and import taxonomy terms.
-   Generate taxonomy term descriptions with AI using your own OpenAI API key.
