# Clean Image Metadata

A WordPress must-use plugin that automatically removes all metadata (EXIF, IPTC, XMP, etc.) from uploaded images. This plugin helps protect user privacy by stripping potentially sensitive information from image files upon upload.

## Features

-   Automatically cleans metadata from all newly uploaded images
-   Processes images during the WordPress upload workflow
-   Cleans metadata from regenerated thumbnails and image sizes
-   Works silently in the background with no configuration needed
-   Helps comply with privacy regulations by removing potentially sensitive data

## Use Cases

-   Privacy-focused websites that handle user-uploaded content
-   Organizations with strict data protection requirements
-   Media libraries where image metadata could contain sensitive information
-   Any WordPress site concerned with minimizing unnecessary data storage
